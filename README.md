# Task 01: Personal Portfolio Website 

This is a clean, modern, and beautiful Personal Portfolio website. I created this project as **Task 01** during my web development internship at **Arch Technologies**. The main goal of this website is to show my education, projects, skills, and work experience in a professional way.

## Features

- **Beautiful Dark Theme:** The website uses a premium dark background with sky-blue text colors and a modern glass-like navigation bar.
- **Project Cards:** Dedicated sections that neatly display my top projects (like the Smart To-Do App) with details about my role and the technology used.
- **Interactive Skills Section:** Displays all my technical skills with a smooth animation effect when you hover over them.
- **Complete Bio Details:** Includes clean sections for my Education, Work Experience, Achievements, Testimonials, and Social Links.
- **Fully Responsive Layout:** The website automatically adjusts its size and looks great on all mobile phones, tablets, and desktop screens.

## Tech Stack Used

- **HTML5:** Used for creating the structure and content of the website.
- **CSS3:** Used for colors, fonts, layout styling (Flexbox), hover animations, and sticky navigation.
- **Vanilla JavaScript:** Connected properly to handle background functionality and future interactive features.

## How to Run the Project

You do not need any special software or code compilers to run this project. You only need a simple web browser.

1. Download all the project files (`index.html`, `style.css`, and `script.js`) and keep them inside the same folder.
2. Find the file named `index.html` inside that folder.
3. Double-click on `index.html` (or right-click it and select "Open with" your web browser like Chrome, Edge, Firefox, or Safari).
4. The website will open instantly and run smoothly.
